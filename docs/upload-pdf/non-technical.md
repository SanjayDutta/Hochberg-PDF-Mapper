# Upload PDF

## Overview
Upload PDF lets users add a PDF document to the application so they can map fields on top of it. It is the starting point of the full workflow.

## Purpose
This feature exists to make onboarding simple and fast. Instead of navigating technical setup steps, users can drag a file or choose it directly and start working in seconds.

## How it works
1. The user drags and drops a PDF file, or clicks the upload area to select one from their device.
2. The app checks basic rules before proceeding: the file must be a PDF and must be within the allowed size limit.
3. If validation passes, the app reads the file and prepares it for editing.
4. A loading state is shown while processing so users know the app is working.
5. After processing, the user is redirected into the editor view where the PDF is displayed and ready for field mapping.

## User Benefits
Upload PDF reduces friction at the first step and prevents common mistakes early, such as wrong file type or oversized files. It also gives clear feedback during upload, so users can confidently continue to field mapping without confusion.

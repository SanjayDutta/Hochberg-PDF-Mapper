# Variable Placement

## Overview
Variable Placement is the feature that lets users put editable fields directly on top of PDF pages.

## Purpose
This feature exists so users can define exactly where information should appear in a document. Instead of manually explaining positions, they can place fields visually on the page and keep the mapping accurate.

## How it works
1. Users choose a field type from the left panel (for example text, number, dropdown, or checkbox).
2. They drag and drop the field onto the PDF page where they want it.
3. A popup appears, where they enter key details such as key and label.
4. The field is shown on top of the PDF and can be resized or repositioned as needed.
5. Clicking an existing field opens editing again, so users can update values later.
6. If a field is no longer needed, users can remove it using the delete icon on that field.
7. The system also supports undo and redo, making it safer to experiment and fix mistakes quickly.

## User Benefits
Users get faster setup, better positional accuracy, and fewer manual errors. The visual workflow makes field mapping easier to understand, even for non-technical users, while still supporting advanced configuration when needed.
